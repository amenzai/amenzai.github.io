os模块

os模块提供与操作系统相关的方法。

API

os.EOL

os.EOL属性是一个常量，返回当前操作系统的换行符（Windows系统是\r\n，其他系统是\n）。

os.arch()

os.arch方法返回当前计算机的架构。

os.tmpdir()

os.tmpdir方法返回操作系统默认的临时文件目录。
