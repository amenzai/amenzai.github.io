react 的一些思考：

  • react：声明式开发
  • jquery：命令式编程
  • 可以与其它框架共存：#root标签之外的 DOM 不受 react 管理
  • 单向数据流：react 不提倡子组件更改父组件数据
  • 视图层框架
  • 函数式编程
